Turns the Another WordPress Classified Plugin listing view into a Pinterest style grid.

== Description ==

Just install and activate, and the main listing of AWPCP will become a Pinterest style grid view. Options and settings coming soon, if people want it.

== Installation ==


1. Install plugin, like you do, ya know?
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Have some homemade Mac and Cheese
